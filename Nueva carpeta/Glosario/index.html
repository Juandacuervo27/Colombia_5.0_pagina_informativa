SENA Tech Glossary

Free/Proprietary Software & Marketing in the Video Game Industry

Software
Open Source
Código Abierto

Software whose source code is publicly available for anyone to inspect, modify, and enhance.

Software
Proprietary Software
Software Propietario

Computer tools where the legal owner retains exclusive rights to use, modify, and distribute under closed licenses.

Software
Free Software
Software Libre

Software that respects users' freedom to run, copy, distribute, study, change, and improve the program.

Software
Source Code
Código Fuente

A collection of lines of text containing the steps a computer must follow, written in a human-readable programming language.

Software
Copyleft
Copyleft / Izquierdo de Autor

A legal practice that uses copyright laws to ensure that modified copies of a software remain free.

Software
EULA (End-User License Agreement)
Acuerdo de Licencia de Usuario Final

A legal contract between the software manufacturer and the user that stipulates the conditions and restrictions of the product's use.

Software
Vendor Lock-in
Dependencia del Proveedor

A situation where a customer finds themselves trapped in a closed software ecosystem due to high costs or difficulties migrating to another.

Software
Fork
Bifurcación

The creation of an independent development project from a copy of an existing software's source code.

Software
Repository
Repositorio

A centralized site where the source code and version history of a project are stored and managed in an organized manner.

Software
DRM (Digital Rights Management)
Gestión de Derechos Digitales

Access control technologies used by publishers and copyright owners to limit the use of digital media or devices.

Software
Freeware
Software Gratuito

Software distributed at no monetary cost, but which keeps its code closed and does not allow modifications.

Software
Shareware
Software de Evaluación

Proprietary software distributed for free with time or feature limitations so the user can evaluate it before purchasing.

Software
Permissive License
Licencia Permisiva

A free software license that imposes minimal restrictions on how it can be redistributed, allowing its use in proprietary software.

Software
Upstream
Flujo de Origen

The direction toward the main repository where bug fixes or improvements from derivative projects are sent.

Software
Community-driven
Impulsado por la Comunidad

Software projects whose direction, technical support, and development are led by volunteers rather than a single corporation.

Marketing
Free-to-Play (F2P)
Gratuito para Jugar

A business model where the base game is downloaded for free, attracting a critical mass, but is monetized through in-game purchases.

Marketing
Microtransactions
Microtransacciones

A commercial model where users acquire virtual goods (cosmetics, advantages) through low-monetary-value transactions.

Marketing
Season Pass
Pase de Temporada

A pre-sale strategy for future and current downloadable content (DLC) bundled at a promotional discount.

Marketing
Early Access
Acceso Anticipado

A marketing and funding strategy where the public pays to play an alpha/beta version to provide feedback to the developer.

Marketing
Publisher
Distribuidor / Editor

A company responsible for funding, manufacturing, promoting, and marketing video games created by development studios.

Marketing
Indie Game
Videojuego Independiente

Games developed by individuals or small groups without the financial support or direct control of a major publisher.

Marketing
AAA (Triple A)
Triple A

An informal classification for games from major publishers with massive development and, above all, marketing budgets.

Marketing
User Acquisition (UA)
Adquisición de Usuarios

Digital marketing processes and campaigns aimed at acquiring new players through paid ads and app store optimization.

Marketing
Target Audience
Público Objetivo / Target

A specific demographic and psychographic profile of players toward whom design and advertising efforts are directed.

Marketing
Influencer Marketing
Marketing de Influenciadores

A key strategy that uses streamers and content creators (Twitch/YouTube) to provide organic visibility to a launch.

Marketing
Teaser Trailer
Tráiler de Avance

A short video campaign to generate anticipation (hype) by revealing very few details about gameplay or story.

Marketing
Beta Testing
Prueba Beta

An open or closed development phase used as a marketing hook to test servers and consolidate the pre-existing community.

Marketing
Hype
Expectativa Desmedida

A communication strategy aimed at creating intense public expectation or excitement around a video game prior to its release.

Marketing
Retention Rate
Tasa de Retención

A marketing metric that measures the percentage of players who continue to interact with the game after a fixed period (Day 1, 7, 30).

Marketing
Loot Box
Caja de Botín

An in-game randomized monetization mechanism where the user pays for surprise digital rewards without knowing the exact contents.
