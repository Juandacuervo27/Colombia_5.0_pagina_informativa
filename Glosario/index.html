Glosario Tecnológico SENA

Software Libre/Cerrado & Marketing en la Industry de Videojuegos

Software
Open Source
Código Abierto

Software cuyo código fuente está disponible públicamente para que cualquiera pueda inspeccionarlo, modificarlo y mejorarlo.

Software
Proprietary Software
Software Propietario

Herramientas informáticas donde el dueño legal retiene los derechos exclusivos de uso, modificación y distribución bajo licencias cerradas.

Software
Free Software
Software Libre

Software que respeta la libertad de los usuarios para ejecutar, copiar, distribuir, estudiar, cambiar y mejorar el programa.

Software
Source Code
Código Fuente

Conjunto de líneas de texto con los pasos que debe seguir la computadora, escrito en un lenguaje de programación comprensible por humanos.

Software
Copyleft
Copyleft / Izquierdo de Autor

Práctica legal que consiste en utilizar las leyes de derechos de autor para garantizar que las copias modificadas de un software sigan siendo libres.

Software
EULA (End-User License Agreement)
Acuerdo de Licencia de Usuario Final

Contrato legal entre el fabricante del software y el usuario que estipula las condiciones y restricciones de uso del producto.

Software
Vendor Lock-in
Dependencia del Proveedor

Situación en la que un cliente se ve atrapado en un ecosistema de software cerrado debido a los altos costos o dificultades para migrar a otro.

Software
Fork
Bifurcación

Creación de un proyecto de desarrollo independiente a partir de una copia del código fuente de un software existente.

Software
Repository
Repositorio

Sitio centralizado donde se almacena y gestiona de manera organizada el código fuente y el historial de versiones de un proyecto.

Software
DRM (Digital Rights Management)
Gestión de Derechos Digitales

Tecnologías de control de acceso usadas por editoriales y propietarios de derechos de autor para limitar el uso de medios o dispositivos digitales.

Software
Freeware
Software Gratuito

Software que se distribuye sin costo monetario, pero que mantiene su código cerrado y no permite modificaciones.

Software
Shareware
Software de Evaluación

Software propietario que se distribuye gratis con limitaciones de tiempo o funciones para que el usuario lo evalúe antes de comprarlo.

Software
Permissive License
Licencia Permisiva

Licencia de software libre que impone mínimas restricciones sobre cómo se puede redistribuir, permitiendo su uso en software cerrado.

Software
Upstream
Flujo de Origen

Dirección hacia el repositorio principal donde se envían las correcciones de errores o mejoras desde los proyectos derivados.

Software
Community-driven
Impulsado por la Comunidad

Proyectos de software cuyo rumbo, soporte técnico y desarrollo están liderados por voluntarios en lugar de una corporación única.

Marketing
Free-to-Play (F2P)
Gratuito para Jugar

Modelo de negocio donde el juego base se descarga gratis, atrayendo masa crítica, pero se monetiza con compras internas.

Marketing
Microtransactions
Microtransacciones

Modelo comercial donde los usuarios adquieren bienes virtuales (cosméticos, ventajas) con transacciones de bajo valor monetario.

Marketing
Season Pass
Pase de Temporada

Estrategia de preventa de contenido descargable (DLC) futuro y actual agrupado con un descuento promocional.

Marketing
Early Access
Acceso Anticipado

Estrategia de marketing y financiamiento donde el público paga por jugar una versión alfa/beta para dar feedback al desarrollador.

Marketing
Publisher
Distribuidor / Editor

Empresa encargada de financiar, fabricar, promocionar y comercializar los videojuegos creados por los estudios de desarrollo.

Marketing
Indie Game
Videojuego Independiente

Juegos desarrollados por individuos o grupos pequeños sin el apoyo financiero ni el control directo de una gran distribuidora.

Marketing
AAA (Triple A)
Triple A

Clasificación informal para juegos de grandes distribuidores con presupuestos masivos de desarrollo y, sobre todo, de marketing.

Marketing
User Acquisition (UA)
Adquisición de Usuarios

Procesos y campañas de marketing digital orientadas a captar nuevos jugadores mediante anuncios pagados y optimización en tiendas.

Marketing
Target Audience
Público Objetivo / Target

Grupo demográfico y de perfil psicográfico específico de jugadores a los que se dirigen los esfuerzos de diseño y publicidad.

Marketing
Influencer Marketing
Marketing de Influenciadores

Estrategia clave que utiliza a streamers y creadores de contenido (Twitch/YouTube) para dar visibilidad orgánica a un lanzamiento.

Marketing
Teaser Trailer
Tráiler de Avance

Campaña corta de video para generar expectativa (hype) revelando muy pocos detalles del gameplay o la historia.

Marketing
Beta Testing
Prueba Beta

Fase de desarrollo abierta o cerrada utilizada como gancho publicitario para testear servidores y afianzar la comunidad previa.

Marketing
Hype
Expectativa Desmedida

Estrategia de comunicación orientada a crear una intensa expectación o emoción pública en torno a un videojuego antes de su estreno.

Marketing
Retention Rate
Tasa de Retención

Métrica de marketing que mide el porcentaje de jugadores que continúan interactuando con el juego tras un periodo fijo (Día 1, 7, 30).

Marketing
Loot Box
Caja de Botín

Mecanismo de monetización in-game aleatorio donde el usuario paga por recompensas digitales sorpresa sin conocer el contenido exacto.
